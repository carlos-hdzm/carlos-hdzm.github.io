About Me

Hello! I'm Carlos, a software developer based in Reno, NV.

My professional experience has a full-stack foundation with a frontend focus. I enjoy transforming data into meaningful visualizations and user interfaces. I have great attention to detail and a strong understanding of user experience principles. I love debugging and solving complex problems. All of these aspects of my work drive me to continuously improve and innovate.

My educational background is in Physics and Applied Mathematics, but from early on I was interested in software development as a means to build useful tools.

I started learning HTML, CSS, and JavaScript in middle school, building simple websites with forms, images, and simple interactions. I remember learning about CSS Sprites and how they could be used to make rounded corners (before CSS3 introduced them). I built a simple HTML page to show my class schedule every semester in high school, which I would send to my classmates.

I also experimented with VB.NET, finding it very useful to build Windows desktop applications. I learned Python in undergrad, which was useful for my Master's thesis, and C/C++ in grad school.

I'm always eager to learn new technologies and improve my skills. That's why I enjoy leaning about the latest Web standards and features, and I'm open to trying new tools for the purposes they may serve.
